üí° Why EasyCall Was Created

EasyCall was born from a real and personal need. My mother-in-law, now 90 years old, has early-stage senile dementia that is affecting her fine motor skills. This prevents her from using a smartphone normally: touchscreens are too small for her, menus too complicated, and gestures too precise.

We tried special phones designed for elderly people, but even those couldn't adapt to her needs.

That's why I decided to develop EasyCall: an app specifically designed so that people with motor or cognitive difficulties can call their loved ones with a single touch. Now she's very happy because she has regained the autonomy to call her loved ones whenever she wants.

Although I developed the app with her in mind, I use it myself every day. Despite having no problem using a smartphone, EasyCall is faster, simpler and more effective for calling my regular contacts than searching for them in my address book or marking them as favorites.

One touch and I call. That simple.

‚öôÔ∏è How to Configure EasyCall

1. Save the contacts you want to see in EasyCall

Using your phone's contacts app, save the contacts you want to select in EasyCall in your address book, and preferably save the photo you want to display for that contact. If the selected contact has no photo, the contact's name will be displayed on the call button. If a contact has multiple phone numbers, EasyCall will use the first phone number saved in the contact.

2. Access settings

Open EasyCall and tap the wrench icon üîß

3. Configure the screen

Select the number of photos horizontally (2-8) and vertically (1-4). Adapt the display to the user's needs. The arrangement of photos on the screen will adapt to the number of selected contacts, but at most, each screen will display the number of photos horizontally and vertically that has been selected.

On this screen you can also enable or disable the display of the configuration icon (the wrench). By default it is enabled and displayed in the upper left corner of the main screen. If you disable it, the icon will disappear but the functionality will remain in the same place: to access the configuration, simply long-press for more than two seconds in the same area that is initially displayed (sometimes the first photo overlaps that area and the press must include the upper left corner of the photo). This option is especially useful to prevent users with cognitive difficulties from accidentally accessing the configuration and leaving the main screen.

4. Add contacts

Tap "Add contact" and select contacts from your address book. EasyCall will use the contact's photo and their first phone number. By tapping on the contact's name you can select another contact for that position. To the left of the contact's position number there is an icon that when held down allows you to move that contact to another position in the list using drag&drop. When finished configuring, save the configuration, and you will return to the main EasyCall page.

üì± Landline Mode (Launcher)

EasyCall can be set as the phone's launcher (home app). Additionally, the screen stays always on while the app is open.

This allows you to leave the phone permanently plugged in on a table, with the screen active and ready to call at any moment with a single touch, just like a landline phone.

Other Features:

Once some contacts are configured, the configuration access icon, a wrench, will appear in the upper left corner of the screen. But in the configuration screen you can disable the display of this icon. In reality the configuration access functionality remains in the same place but is not visible and to access it you must long-press for more than two seconds in the same area that is initially displayed (sometimes the first photo overlaps the configuration access area and the press must include the upper left corner of the photo).

On some phone models, for the hands-free functionality to activate automatically it will be necessary to allow Restricted Settings for the EasyCall application. It is configured in Settings/Applications/EasyCall, but this may vary depending on each smartphone model.

‚ùì Frequently Asked Questions

What devices does EasyCall work on?
EasyCall works on any Android smartphone or tablet. The app automatically adapts to your device's screen size.
Can I use my own contact photos?
Yes, EasyCall uses the photos you already have saved in your contacts. You just need to have contacts with photos in your phone's contacts app. But if you select a contact without a photo, the contact's name that you have saved will be displayed.
Does automatic hands-free work on all phones?
The automatic hands-free activation feature does not work on all phone models, as it depends on each manufacturer. In some cases it can be fixed by granting accessibility permissions to the app, but in other cases the manufacturer does not allow automatic hands-free activation.
Can I set the app as a permanent home screen?
Yes, EasyCall can be set as the phone's launcher (home app). This is ideal for turning the device into a digital landline phone. To exit when set as launcher, press the phone's power button.
How many contacts can I add?
You can add as many contacts as you need. If you select more contacts than fit on one screen, additional screens will be created automatically. A navigation button will allow you to move between them.
Does EasyCall store my data on any external server?
No. EasyCall does not store any contact data on its own servers or anywhere external. The app only accesses contact data that you already have saved in your own phone. All configuration is stored locally on your device.
Do I need technical knowledge to configure it?
No. The configuration is very simple and intuitive. Any family member or caregiver can configure the contacts in a few minutes. Once configured, the end user will only see the photos and will be able to call with a simple touch.
Can I uninstall other phone apps if I use EasyCall as launcher?
Although EasyCall can function as a launcher, it does not restrict access to other apps nor is it necessary to uninstall them. It will simply make EasyCall the app that is displayed when turning on the phone or pressing the home button.
What happens if I change a contact's data or photo after configuring it?
EasyCall maintains the contact's data at the time it was saved in the app. If you want to change the contact's photo and contact data, you must remove the contact from EasyCall's configuration and add it again so that it updates the photo and/or phone number.
What happens if I delete a configured contact from my address book?
If you delete a contact from your address book, that contact's data in EasyCall will remain. If you want to also remove it from EasyCall, you will need to access EasyCall's configuration again to remove that contact from the app or replace it with another.
I disabled the configuration tool display and can't access the configuration anymore?
Access to the configuration remains in the same position even though it is not visible, but now to access it you must press for more than two seconds in that area. Sometimes the first photo on the screen overlaps the configuration access area and the press must include the upper left corner of the photo.
